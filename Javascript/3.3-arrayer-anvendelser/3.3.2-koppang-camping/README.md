# Koppang camping

## Instruksjoner
Du skal lage et nettsted basert på [koppangcamping.no](https://www.koppangcamping.no/no/koppang-camping/)

- Del 1: Hovedside `index.html` med generell informasjon om campingplassen. Noen bilder fra Glomma og stedet vises.
- Del 2: Hytter `hytter.html` med informasjon og bilder av de 3 ulike hyttetypene.
- Del 3: Bookingside hvor en bruker kan booke en hytte ved hjelp av et skjema.
